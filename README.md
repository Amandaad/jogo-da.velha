# Jogo da Velha Adson

Jogo da velha desenvolvido com `HTML`, `CSS` e `JavaScript`, com visual personalizado e peças representadas por dois bonecos.

## Visão geral

O projeto foi criado para funcionar de forma simples no navegador, sem necessidade de instalação de dependências. Basta abrir o arquivo principal para começar a jogar.

## Funcionalidades

- Tabuleiro interativo 3x3
- Alternância automática entre os dois jogadores
- Dois bonecos diferentes como peças do jogo
- Mensagem de turno e mensagem de vitória
- Contador de vitórias e empates
- Botão para iniciar uma nova rodada
- Botão para zerar o placar

## Estrutura do projeto

- `index.html` - estrutura da interface
- `style.css` - estilos e aparência visual
- `script.js` - lógica do jogo e interações
- `README.md` - documentação do projeto

## Como executar

1. Abra a pasta do projeto.
2. Dê dois cliques em `index.html` ou abra esse arquivo no navegador.
3. Comece a jogar normalmente com duas pessoas no mesmo computador.

## Tecnologias utilizadas

- `HTML5`
- `CSS3`
- `JavaScript`

## Observações

- As imagens dos bonecos são carregadas por links externos.
- Para que os bonecos apareçam no jogo, a internet precisa estar disponível.
